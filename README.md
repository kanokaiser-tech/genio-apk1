# Genio Revendedores - APK

## Instrucciones paso a paso

### 1. Crear repositorio en GitHub
1. Andá a https://github.com/new
2. Nombre: `genio-apk`
3. Dejalo PUBLICO
4. NO marques "Add a README"
5. Click en "Create repository"

### 2. Subir los archivos
1. En la pagina del repo nuevo, click en "uploading an existing file"
2. Descomprimi el ZIP que te di
3. Arrastra TODOS los archivos y carpetas (android/, .github/) al area de subida de GitHub
4. Click en "Commit changes"

IMPORTANTE: Tenes que subir TODO incluyendo:
- android/
- .github/
- README.md

### 3. Ejecutar el workflow
1. Andá a la pestaña "Actions" de tu repo
2. Click en "Build APK"
3. Click en "Run workflow" (boton gris a la derecha)
4. Click en "Run workflow" (verde)
5. Espera 3-5 minutos

### 4. Descargar la APK
1. Cuando termine (circulo verde), click en el workflow
2. Abajo en "Artifacts", click en "genio-revendedores-apk"
3. Se descarga un ZIP, adentro esta la APK

### 5. Instalar en el celular
1. Pasá el .apk al celular
2. Tocalo para instalar
3. Acepta "Fuentes desconocidas" si pide
